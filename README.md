# Research Timeline — Structure-Centered Reasoning in LLMs  
# 연구 타임라인 — LLM의 구조 중심 추론

This repository is a public research timeline that documents the prior definition, naming, and structural formulation of concepts related to structure-centered reasoning, predictive consistency, and cognitive alignment in large language models (LLMs).  
This repository is **not** intended for collaboration, implementation, validation, or model evaluation. Its role is to publicly anchor the existence of specific technical terms, frameworks, and reasoning modes at a defined point in time.

본 저장소는 대규모 언어모델(LLM)의 **구조 중심 추론**, **예측 일관성**, **인지 정렬**과 관련된 개념들의 **선행 정의/명명/구조화 정식**을 타임라인 형태로 공개 기록합니다.  
본 저장소는 협업, 구현, 검증, 성능평가를 목적하지 않습니다. 특정 용어·프레임워크·추론 모드의 존재를 특정 시점에 **공개적으로 고정(앵커링)**하는 것이 목적입니다.

## Timeline Items / 타임라인 항목(요약)

1) **Predictive-Consistency Mode (PCM)**  
Core terms: Predictive-Consistency Mode (PCM), Self-Diagnostic Reasoning, Predictive Constraint Loop, Internal Consistency Stabilization  
구성 용어: PCM, 자기진단 추론, 예측 제약 루프, 내부 일관성 안정화

2) **NorthStar Cognitive Alignment Framework (NCAF)**  
Core terms: Cognitive Reset / Observe / Deconstruct / Reconstruct, Cognitive Alignment Flow  
구성 용어: 인지 리셋 / 관찰 / 해체 / 재구성, 인지 정렬 흐름

3) **Cognitive Frame Shift & Structure-First Inference**  
Core terms: Frame-Level Transition, Structural Trigger, Structure-First Reference  
구성 용어: 프레임 전이, 구조 트리거, 구조 우선 참조

(Details / 상세는 TERMINOLOGY.md 참조)
