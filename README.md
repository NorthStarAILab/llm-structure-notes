Structural Definition
PCM is a reasoning mode in which LLM outputs are governed by predictive consistency constraints across inference steps, rather than post-hoc correction or output filtering.
The primary objective is not correctness per se, but structural coherence of the reasoning trajectory itself.
Hallucination is treated not as a data insufficiency problem, but as a failure of internal predictive structure.

Structural Definition
NCAF is a structure-centered cognitive alignment framework describing how reasoning stability is formed and restored.
Alignment is treated as a process-level structural property, not as a reward-optimization outcome.
The framework prioritizes cognitive reset and reconstruction mechanisms over instruction-following behavior.

Under specific structural conditions, LLM responses exhibit discrete frame-level transitions, rather than continuous probabilistic drift.
